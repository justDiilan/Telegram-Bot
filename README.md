# Telegram Support Bot

Простой Telegram-бот на Python, который показывает пользователю клавиатуру с тремя кнопками:

* **🖼 Изображение** – отправляет подготовленное изображение.
* **🎞 Гифка** – отправляет анимированную гифку.
* **🛠 Тех. поддержка** – переводит пользователя в режим техподдержки, в котором его сообщение отправляется оператору.

## Подготовка окружения

1. Склонируйте репозиторий и перейдите в его директорию.
2. Создайте виртуальное окружение и установите зависимости:

   ```bash
   python -m venv .venv
   source .venv/bin/activate  # Windows: .venv\Scripts\activate
   pip install -r requirements.txt
   ```
3. Скопируйте файл `.env.example` в `.env` и заполните значения:

   ```bash
   cp .env.example .env
   ```

   * `TELEGRAM_BOT_TOKEN` — токен вашего бота из @BotFather.
   * `SUPPORT_CHAT_ID` — идентификатор чата, куда будут пересылаться обращения (например, ваш личный ID или ID группы).

## Запуск локально

```bash
python bot.py
```

Бот использует `python-telegram-bot` (версии 20+) и запускается в режиме `long polling`.

## Развёртывание на бесплатный хостинг

Проект ориентирован на развёртывание на хостинг Render, Railway или любой другой, поддерживающий запуск простого Python-приложения.

Пример для [Render](https://render.com/):

1. Создайте новый сервис типа **Web Service**, подключив репозиторий.
2. Введите команду запуска `python bot.py`.
3. Установите переменные окружения `TELEGRAM_BOT_TOKEN` и `SUPPORT_CHAT_ID` в настройках сервиса.
4. Выберите бесплатный тариф (Free). После деплоя бот начнёт отвечать пользователям.

При использовании Railway настройка аналогична: достаточно добавить переменные окружения и команду запуска.

## Настройка техподдержки

* Пользователь нажимает кнопку «🛠 Тех. поддержка» и вводит сообщение.
* Бот пересылает текст запроса, имя, идентификатор пользователя и ID чата на указанный `SUPPORT_CHAT_ID`.
* Пользователь получает подтверждение, что запрос отправлен.

## Дополнительные советы

* Для замены изображений и гифок отредактируйте константы `IMAGE_URL` и `GIF_URL` в файле `bot.py`.
* Чтобы обновить клавиатуру или локализацию — измените значения `IMAGE_BUTTON`, `GIF_BUTTON` и `SUPPORT_BUTTON`.
